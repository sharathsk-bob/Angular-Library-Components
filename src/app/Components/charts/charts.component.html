PieChart

Details for the PieChart section are below.

Please do follow the below given instructions while creating your PieChart component.

  • User can select the number of options (maximum upto 5)
  • User can add the name and value for each data(title:3-15 characters, value:1-1000)
  • User can select the size for the piechart
  • User can select the theme for the piechart

Donut Chart

Details for the Donut section are below.

Please do follow the below given instructions while creating your Donut component.

  • Input/Textarea label should be less than 15 characters.
  • Input text should be less than 50 characters.
  • Textarea text should be less than 50 characters.
  • You can select width of input/textarea from the given options.
  • You can select rounded radius of input/textarea from the given options.
  • You want to show the number of input/textarea characters select yes from the given options.
  • You can select the input/textarea theme.
  • You can select the input/textarea input type.

Bar Chart

Details for the bar chart section are below.

Please do follow the below given instructions while creating your bar chart component.

  • User can select the number of bars for bar chart- from 1 to 5.
  • User can enter title and value for number of selected bars.
  • User can select the size from small, medium or large.
  • User can select the theme of the bar chart from the dropdown.
